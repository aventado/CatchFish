## 概述
这是一个薅区块鱼羊毛的脚本，通过模拟用户注册、激活，来完成模拟用户邀请

## 运行环境要求
- 一个临时邮箱服务：我这地方选用的是[YOPmail](http://www.yopmail.com/zh/)
- Python
- chrome webdriver
- selenium
- iterm2(安装imgcat 工具): 因为要输入验证码，所以选用iterm2，这样可以把验证码图片输出到终端。

## 我的测试环境
- Mac+Python2.7+iTerm2+imgcat
